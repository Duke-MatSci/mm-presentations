What Is MaterialsMine?

MaterialsMine is a web-based data resource designed and built for the materials research community.

The objectives of MaterialsMine include:

  • Curating experimental and simulation data into a structured, machine-readable format
  • Providing an interface for researchers to retrieve materials data
  • Enabling data-driven tools for materials property prediction and design

Materials Knowledge Graph

MaterialsMine data is ultimately stored as a knowledge graph, a multitude of nodes with edges describing the relationships between nodes. Nodes and edges have unique identifiers (URIs), making all relationships unambiguous and machine-readable.

Search & Visualization

To query data within MaterialsMine, a faceted browser enables the selection of parameters of interest.

For users familiar with RDF and the semantic web, MaterialsMine also features a SPARQL endpoint for querying the Knowledge Graph directly.

Analysis & Simulation Tools

A suite of modular tools enable researchers to apply data-driven tools directly to their data.

For example, the Microstructure Characterization & Reconstruction tool provides quantitative descriptors on microscope images (e.g. TEM) to quantify dispersion parameters associated with the image. These descriptors provide a basis for further analysis.
